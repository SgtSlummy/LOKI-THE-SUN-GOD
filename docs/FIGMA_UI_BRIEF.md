# Figma UI Brief

Use this brief if a Figma file is created for the LOKI portal or desktop dashboard.

## Surfaces

- Web portal dashboard for Discord OAuth admins.
- Music mixer with queue, output volume, EQ presets, DJ lock, and current track.
- NPC controls for persona, public channel allowlist, memory deletion/export, and web recommendation settings.
- Activity control page for scheduled events, embedded activity handoff, and portal quests.
- Developer settings page for Discord app, OAuth redirect, LLM provider, Railway, and MCP status.

## Visual Direction

Operational and dense, not a marketing page. Use restrained dark UI, compact controls, strong scan hierarchy, icons for common actions, and stable dimensions for mixer/deck controls.

## Required States

- Read-only viewer.
- Admin editor.
- Missing Discord OAuth.
- Missing OpenAI key.
- Lavalink disconnected.
- Mythos packet pending.
- Memory/web crawler disabled.
